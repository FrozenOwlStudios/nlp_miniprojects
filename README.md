# NLP MINIPROJECTS

This is a repository where I store all my miniprojects related to
natural language processing that do not use LLM.
What you will find here are nltk, antlr or prolog based concepts.
Miniproject are usually contained to single file, unless it's impossible,
so that they can be later easily used as a reference for future more
serious projects.
The overall project is managed by UV as python is the dominant language here,
however they can appear some miniproject that have to be handled outside
UV environment.
In such case those projects will have their own folder with README containing
instructions on how to compile/run them.
